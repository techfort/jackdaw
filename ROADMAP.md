# JackDAW Project Roadmap 🎸

## Core Premise
A collaboration-first "GitHub for Music," not a generic DAW. Focus on the communication loop: **Comment -> Punch-In -> Align -> Resolve.**

## 🟢 Phase 1: Collaboration & Synchronization (In Progress)
- [x] Real-time presence (Cursors & Playhead)
- [x] Project Hub / Issue Tracker
- [x] Time-stamped commenting
- [x] Firestore-backed global state sync

## 🟡 Phase 2: The "Artist Loop" (Next Up)
- [ ] **One-Click Punch-In**: Drop audio at playhead location.
- [ ] **Take Layers**: Non-destructive "Alternatives" on the same track.
- [ ] **Status Workflow**: "Needs Feedback", "In Progress", "Approved".
- [ ] **Track Freezing**: Lightweight preview for heavy projects.

## 🔴 Phase 3: Intelligent Assistance (Future)
- [ ] **Auto-Alignment**: Transient-based cross-correlation to snap snippets to reference tracks.
- [ ] **AI Summaries**: "What changed since I last logged in?"
- [ ] **Phase Alignment Helper**: Suggesting 180° flips for thin-sounding punches.

## ❄️ Low Priority / Deprioritized
- Deep mixing effects (Reverb/Delay).
- Remote recording (Focus is on local recording + upload).
- User signup/login (Stick to anonymous/session-based for now).
